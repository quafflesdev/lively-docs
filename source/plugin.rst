Plugins
=======

Plugins enable developers to extend the platform's functionalities. A plugin can perform many functions.

Functions
---------

1. Search Handler   -   It can act as a hook to handle search query from lively's homepage.
2. Push Events      -   It can push event on behalf of a user.
3. Notify Users     -   It can help user customize notifications as per their use.

Examples
---------

1. Search Handler   -   A plugin which helps user search for all outdoor events.
2. Push Events      -   A plugin that helps user set recurring events.
3. Notify Users     -   A plugin that notifies user about an event with attendess from diverse ethnicity.
